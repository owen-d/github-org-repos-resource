* github-org-repos-resource
libs to talk to github
produces cli tools
* Source
** api_key :: String
** org :: String
* Params
None yet

* Methods
** Check
Reads stdin:
#+BEGIN_SRC JSON
{
  "source": {
    "api_key": "abc123...",
    "org": "LuminalHQ"
  },
  "version": { "ref": "abc123..." }
}
#+END_SRC
stdout (can be empty):
#+BEGIN_SRC JSON
[
  { "ref": "abc123..." }
]
#+END_SRC
** In
destination dir as $1
source + version on stdin
drops json metadata of repos in dest dir
outputs info on stdout:
#+BEGIN_SRC JSON
{
  "version": { "ref": "61cebf" },
  "metadata": [
    { "name": "commit", "value": "61cebf" },
    { "name": "author", "value": "Hulk Hogan" }
  ]
}
#+END_SRC
** Out
Noop
